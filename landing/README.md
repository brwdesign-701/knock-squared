# Knock² Landing Page

A modern, responsive marketing landing page for Knock² - a SaaS product that helps service companies improve customer trust by sending verified technician profiles before they arrive.

## Features

- **Modern Design**: Clean, trustworthy design with custom branding (Primary: #0B2E51, Teal: #39C0C3, Sky Blue: #1D7AD7)
- **Responsive**: Fully mobile-responsive layout
- **Smooth Animations**: Framer Motion animations throughout
- **SEO Optimized**: Complete metadata and Open Graph tags
- **11 Sections**:
  1. Header with navigation
  2. Hero section with CTAs
  3. Value Pillars (4 key benefits)
  4. How It Works (3-step process)
  5. Product Preview
  6. Testimonials
  7. Pricing (3 tiers)
  8. Security features
  9. FAQ (accordion)
  10. Final CTA
  11. Footer

## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS 3**
- **Framer Motion** (animations)
- **Lucide React** (icons)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Building for Production

```bash
npm run build
npm start
```

## Deployment

This landing page can be deployed to Vercel, Netlify, AWS Amplify, or any Node.js hosting platform.

For Vercel:
```bash
vercel
```

## Customization

### Colors
Edit `tailwind.config.ts` to change brand colors.

### Content
All content can be edited directly in the component files in the `components/` directory.
